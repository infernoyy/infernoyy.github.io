github代码管理

代码管理步骤

1、代码仓库优先建立好，git提前安装
2、新建文件夹，右击Git Bash Here
3、克隆远端仓库

git clone https://github.com/infernoyy/BlogCode.git

4、cd 你的文件夹
5、添加、提交、推送

  • git add .
    注：别忘记后面的.，此操作是把Test文件夹下面的文件都添加进来）

  • git commit -m “提交信息”
    （注：“提交信息”里面换成你需要，如“first commit”）

  • git push -u origin master （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）
